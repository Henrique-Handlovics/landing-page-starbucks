☕ Landing Page Starbucks

Landing page inspirada no visual do site da Starbucks, criada com foco em design responsivo, layout moderno e habilidades de Front-end básico.

<hr>

🚀 Sobre o Projeto

Projeto desenvolvido para prática de desenvolvimento Front-end com HTML e CSS, organizando uma página estática que apresenta seções temáticas com elementos visuais e responsivos, tipicamente usadas em projetos de apresentação de marcas ou produtos.

<hr>

🛠️ Tecnologias

HTML5

CSS3

<hr>

🎓 Inspiração

Projeto desenvolvido como parte de estudo prático em Front-end, inspirado por conteúdo educacional de vídeo tutorial e aplicado para consolidar conhecimentos de estruturação, responsividade e design básico de páginas web.
